## NEPALI LYRICS APP (IN PROGRESS..)

>Tools used:

- Nodejs (backend)
- ejs (template)
- Express(Framework)
- Firebase (Database, Authentication)
- JS
- HTML
- CSS
- Bootstrap 
--- 

From the working directory,
```terminal
npm install
```

To start in browser: 
```terminal
node app.js
```
